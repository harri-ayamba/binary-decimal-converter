Binary ⇄ Decimal Converter
A simple web tool that converts between binary and decimal numbers.

Enter a binary number to see its decimal value.

Enter a decimal number to see its binary value.

Validates input to prevent invalid numbers, and gives clear feedback.
Built with HTML, CSS, and vanilla JavaScript—great for practicing number systems and DOM interaction.


Binary uses only 0s and 1s to represent numbers, which is the base‑2 system computers use.
